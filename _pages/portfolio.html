ESICard: Multifunctional Student ID Cards

2019 | BCG Article

Led the technical development of a multifunctional student ID card for the Lichtenbergschule while I was a student there. We created tools for the card issuance process and developed interfaces for the student and library databases. The system remains in active use today.

Reference Database for Mammographic Diagnostic Procedures

2015

Developed an assistive program to support medical staff in selecting appropriate mammographic diagnostic procedures. The program was created for the University Clinic Heidelberg and continues to be actively used today.
